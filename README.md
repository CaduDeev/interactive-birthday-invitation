# Interactive Birthday Invitation Website

This project was developed for my brother's birthday celebration.

The website provides a modern and interactive experience for guests, including personalized invitation links, animated pages, video introductions, and RSVP management.

## Features
- Personalized guest invitation links
- Responsive design
- Video introduction
- Interactive animations
- Guest management system
- RSVP confirmation

## Technologies
- HTML5
- CSS3
- JavaScript
- Firebase

## Purpose
This project was created to improve my web development skills while building a real-world solution for a family event.
